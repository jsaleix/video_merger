# VIDEO MERGER

```
Dans le repertoire courant:
- récupére tout les fichiers vidéos (mp4, mov, mkv)
- les fusionne dans un seul fichier avec un ecart de X secondes entre chaque video
- genere un fichier "timecode.txt" avec dedans le debut de chaque video avec leur nom
    00:00:00 - nom_de_la_video
    00:05:23 - nom_de_la_seconde_video
    etc...
```
